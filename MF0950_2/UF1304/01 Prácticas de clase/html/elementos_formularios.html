Elementos de un formulario:

Tag Description
<form> Defines an HTML form for user input
<input> Defines an input control
<textarea> Defines a multiline input control (text area)
<label> Defines a label for an <input> element
<fieldset> Groups related elements in a form
<legend> Defines a caption for a <fieldset> element
<select> Defines a drop-down list
<optgroup> Defines a group of related options in a drop-down list
<option> Defines an option in a drop-down list
<button> Defines a clickable button
<datalist> Specifies a list of pre-defined options for input controls
<output> Defines the result of a calculation

Elemento <input>

El elemento <input> puede mostrarse de múltiples formas, dependiendo del tipo de atributo.

<input name="firstname" type="text">

Si se omite el atributo type, el valor por defecto es type: "text".

Elemento <select>

El elemento <select> define una lista desplegable:

<select name="cars">
  <option value="volvo">Volvo</option>
  <option value="saab">Saab</option>
  <option value="fiat" selected>Fiat</option>
  <option value="audi">Audi</option>
</select>

El elemento <option> define una opción que puede seleccionarse.

Por defecto está seleccionado el primer item de la lista. Para predeterminar una opción, utilizar el atributo "selected"

Elemento <textarea>

El elemento <textarea> define un campo multilínea (área de texto)

rows: atributo para definir el número de líneas visibles del área de texto.

cols: atributo para definir el número de columnas visibles del área de texto.

En lugar de rows y cols se puede usar css (width, height) para definir el tamaño.

Elemento HTML5 <datalist>

<datalist> especifica una lista de opciones pre-definida para el elemento <input>.

Los usuarios ven una lista desplegable de opciones predefinidas de datos.

La lista de atributos del elemento <input>, debe de referise al atributo id del elemento <datalist>.
