Tablas HTML

Etiquetas que definen una tabla

Una tabla HTML se define con la etiqueta <table> tag.

Cada tabla se define con la etiqueta <tr>. La cabecera se define con <th>. Por defecto, los encabezados son centrados y en negrita. Una celda se define con la etiqueta <td>


Etiqueta Descripción
<table> Define una tabla
<th> Define una celda de cabecera
<tr> Define una fila
<td> Define una celda en la tabla
<caption> Define un titulo
<colgroup> Define una agrupación de columnas para formatearlas
<col> Especifica propiedades de una columna dentro del un colgroup
<thead> Agrupa el contenido de cabecera de una tabla
<tbody> Agrupa el contenido del cuerpo de una tabla
<tfoot> Agrupa el contenido del pie de una tabla

El atributo Scope

El atributo scope provee un mecanismo para indicar explícitamente a qué celdas afecta una celda de encabezado. Solo afecta a las etiquetas <th>

El atributo scope especifica si una celda de cabecera es de fila, columna o de grupo de columnas o filas. No tiene efectos visuales en los navegadores, pero se puede utlizar para los lectores de pantalla.

Atributos colspan y rowspan

El atributo colspan define el número de columnas que una celda debe ocupar. El atributo rowspan especifica el número de filas que una celda debe ocupar.

Etiquetas colgroup y col

La etiqueta <colgroup> especifica un grupo de una o más columnas en una tabla para dar formato. La etiqueta <col> especifica las propiedades de columna para cada columna de un <colgroup>. Es útil para aplicar estilos a columnas completas sin tener que repetir los estilos pra cada celda..
