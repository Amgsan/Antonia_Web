Formato HTML

El formato del texto, es el marcado que se aplica a un texto para añadirle datos de forma

Al texto resultante se le conoce como texto formateado, texto con estilos o texto enriquecido

Encabezados de texto

Las máquinas de búsqueda utilizan los encabezados para indexar la estructura y el contenido de las páginas web

Los usuarios hojean las páginas por sus encabezados. Es importante utilizar los encabezados para mostrar la estructura del documento

Los encabezamientos <h1> deben utilizarse para encabezados principales, seguidos de encabezamientos <h2>, después <h3>; y así sucesivamente

Nota: utilizar encabezados HTML solo para encabezados. No usar los encabezados para texto GRANDE o negrita.

Etiqueta Descripción Ejemplo Utilización

Esto es un encabezado de tipo 1

 <h1>Esto es un encabezado de tipo 1</h1>

Formateo de elementos HTML

HTML tiene elementos especiales para definir texto con un siginificado especial.

HTML utiliza elementos como <b> e <i> para formatear el texto como negrita o cursiva.

Etiqueta Descripción Ejemplo Utilización
<b> Describe texto en negrita Esto es un texto en negrita <b>Esto es un texto en negrita</b>
<strong> Describe texto importante o palabra clave Esto es un texto importante <strong>Esto es un texto importante</strong>
<i> Describe texto en cursiva Esto es un texto en cursiva <i>Esto es un texto en cursiva</i>
<em> Describe texto enfatizado Esto es un texto enfatizado <em>Esto es un texto enfatizado</em>
<mark> Describe texto marcado Esto es un texto marcado <mark>Esto es un texto marcado</mark>
<small> Describe texto pequeño Esto es un texto más pequeño <small>Esto es un texto pequeño</small>
<del> Describe texto borrado Esto es un texto borrado <del>Esto es un texto borrado</del>
<ins> Describe texto añadido Esto es un texto añadido <ins>Esto es un texto añadido</ins>
<sub> Describe texto subíndice Esto es un texto subíndice <sub>Esto es un texto subíndice</sub>
<sup> Describe texto superior Esto es un texto superior <sup>Esto es un texto superior</sup>
<s> Describe texto tachado Esto es un texto tachado <s>Esto es un texto tachado</s>
<u> Describe texto subrayado Esto es un texto subrayado <u>Esto es un texto subrayado</u>

Nota: Los navegadores sacan <strong> como <b>, y <em> como <i>. Sin embargo, hay diferencia en el significado de esas etiquetas: <b> y <i> definen texto en negrita y en cursiva, pero <strong> y <em> significa que el texto es "importante".
