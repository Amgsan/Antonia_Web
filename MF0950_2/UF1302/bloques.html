Etiquetas que citan el texto

Las abreviaturas de marcado <abbr> pueden proporcionar información útil a los navegadores, sistemas de traducción y motores de búsqueda. Utiliza el atributo title para definir el significado de la abreviación.

La etiqueta de marcado <cite> hace referencia a un trabajo creativo como un libro, una película, un programa de televisión o una canción, no el nombre de una persona.

La etiqueta de marcado <blockquote> define una cita larga que se representa como un elemento de bloque. No genera automáticamente comillas alrededor del contenido.

La etiqueta de marcado <q> define una cita corta en línea. Marca las citas a otros autores o documentos. Ggenera automáticamente comillas alrededor del contenido.

La etiqueta de marcado <dfn> define un término y debe contener el atributo title para la definición / explicación del término dentro. Se usa para el navegador.

Elementos que agrupan el texto

El <div> es un elemento que actúa como un contenedor de otros elementos. No tiene necesariamente atributos, pero <style>, <class> e <id> son habituales. Cuando se utiliza junto con CSS, puede ser utilizado para dar estilo a un bloque de contenido.

El elemento <span> a menudo se utiza como contenedor para agrupar un texto en línea. No tiene necesariamente atributos, pero <style>, <class> e <id> son habituales para dar estilos CSS a un trozo de texto.

Etiquetas de código de computadora

  • La etiqueta <code> define un fragmento de código de ordenador.
  • La etiqueta <kbd> representa comandos de entrada del usuario.
  • La etiqueta <samp> representa la salida de un programa o sistema informático.
  • La etiqueta <var> se utiliza como expresión matemática o variable de programación.
