¿Que es Multimedia?

Los elementos multimedia pueden tener diferentes formatos. Ejemplos son; imágenes, música, sonido, vídeos, películas, animaciones, etc. Las páginas web a menudo contienen elementos multimedia de diferentes tipos y formatos.



HTML Vídeo


Etiqueta Descripción
<video> Define un vídeo
<source> Define la fuente para los elementos multimedia, ya sea video o audio
<track> Define el fichero de subtítulos
<audio> Define un audio
<iframe> Define un marco en el que sacar una fuente externa
<object> Define un elemento embebido en un documento HTML. Se utiliza este elemento para incorporar multimedia embebida en la pagina web (like audio, video, Java applets, ActiveX, PDF, y Flash) u otra página web. Se ha quedado relegado para ficheros tipo .swf
<embed> Define un contenedor para una aplicación externa o contenido interactivo.

<video width="320" height="240" controls autoplay loop poster="img/sencha-touch.jpg" preload="auto">
<source src="video/mov_bbb.mp4" type="video/mp4">
<source src="video/mov_bbb.webm" type="video/webm">
<source src="video/mov_bbb.ogg" type="video/ogg">
<track src="subtitulos_en.vtt" kind="subtitles" srclang="en" label="English">
</video>
