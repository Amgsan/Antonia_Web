¿Que es CSS?

HTML es el lenguaje estándar de marcado para crear páginas web.

  • CSS significa Cascading Style Sheets (hojas de estilo en cascada)
  • CSS describe  como se van a mostrar los elementos HTML en pantalla, papel u otros medios
  • CSS  ahorra mucho trabajo. Puede controlar el diseño de varias páginas a la vez
  • Las hojas de estilo externas se almacenan en ficheros CSS

Estructura básica de un documento HTML

Ejemplo

<!DOCTYPE html>
<html ="es-ES">
<head>
<title>Page Title</title>
<meta charset="utf-8">
</head>
<body>
<h1>My First Heading</h1>
<p>My first paragraph.</p>
</body>
</html>

Explicación de las etiquetas básicas

  1. La declaración <!DOCTYPE html> define que el documento es HTML5
  2. El elemento <html> es el elemento raiz de una página HTML
  3. El atributo lang="es-ES" especifica el idioma del documento y va dentro de la etiqueta
  4. El elemento <head> contiene meta información sobre el documento
  5. El elemento <title> define el titulo del documento
  6. El elemento <meta charset="UTF-8"> define el juego de caracteres
  7. El elemento <body> contiene el contenido visible de la página
  8. El elemento <h1> define una cabecera principal
  9. El elemento <p> define un párrafo

Etiquetas HTML

Las etiquetas HTML son los nombres de los elementos contenidos entre paréntesis angulares:

<tagname>aquí va el contenido...</tagname>

Etiquetas HTML
normalmente vienen en pares como <p> y </p>
Las etiquetas HTML
tienen una etiqueta de inicio y una de fin
La etiqueta de fin
se escribe como una etiqueta de incio pero precedida de una barra inclinada antes del nombre de la etiqueta

Consejo: La etiqueta de inicio también se llama etiqueta de apertura, y la de final etiqueta de cierre.

Estructura de una Página HTML

A continuación se muestra una visualización de una estructura de página

<html>

<head>

<link rel="stylesheet" href="css/estilos.css" type="text/css">

<style> h1 {font-size:14px; color:Blue;}</style>

</head>

<body>

<h1>This is a heading</h1>

<p>This is a paragraph</p>

<p>This is another paragraph</p>

</body>

</html>

Nota: sólo el contenido dentro de la sección <body> (la zona blanca de arriba) sale en el navegador
